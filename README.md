# ⚽ 축구 유망주 탐색 Interactive Dashboard

Football Manager 선수 데이터(약 16만 명)를 분석하여 유망주를 찾는 interactive 웹 대시보드입니다.

## 🎯 주요 기능

### 1. 포지션별 맞춤 필터 시스템
- **포지션 선택**: 골키퍼, 수비수, 미드필더, 공격수
- **포지션별 핵심 스텟 필터**: 
  - 🤚 **골키퍼**: 반사신경, 핸들링, 일대일, 박스장악, 킥력, 민첩성
  - ⚔️ **수비수**: 대인마크, 태클, 헤딩, 포지셔닝, 예측력, 근력, 스피드
  - 🎯 **미드필더**: 패스, 시야, 기술, 볼터치, 스태미나, 활동량, 판단력
  - 🚀 **공격수**: 골결정력, 드리블, 스피드, 가속력, 오프더볼, 침착성, 기술
- **나이 범위**: 젊은 선수 위주로 필터링
- **국적**: 상위 20개 국가별 필터

### 2. 유망주 평가 시스템
- **종합 능력치**: 기술(40%) + 정신(35%) + 신체(25%)
- **잠재력 점수**: 나이에 따른 가중치 적용
  - 18-21세: 1.5배
  - 22-25세: 1.2배
  - 26-29세: 1.0배
  - 30세 이상: 0.5배
- **포지션 특화 점수**: 각 포지션별 중요 능력치 평가

### 3. Interactive 시각화 (드릴다운 방식)
- **🎯 선수 발굴 탭 (Scatter Plot)**:
  - 축 선택 기능 (나이, 스피드, 패스, 드리블 등)
  - 포지션별 자동 최적화 (공격수: 골결정력 vs 스피드)
  - 드래그로 줌인/아웃 가능
  - 선수 클릭 시 5개 대분류 레이더 차트 즉시 표시
  
- **📊 선수 비교 탭 (Parallel Coordinates)**:
  - 상위 N명 유망주 동시 비교
  - 각 축(능력치)별 드래그 필터링
  - 2-3명 선택하여 레이더 차트 직접 비교
  - 포지션별 핵심 스텟 바 차트
  
- **🏆 상위 유망주 탭**:
  - 상위 20명 바 차트
  - 상세 리스트 및 CSV 다운로드
  
- **📈 포지션별 분석 탭**:
  - 포지션별 평균 점수 및 분포
  - 박스 플롯으로 능력치 비교
  
- **👤 선수 프로필 탭**:
  - 5개 대분류 레이더 차트 (Attacking, Defending, Technical, Mental, Physical)
  - 강점/약점 자동 분석
  - 포지션별 핵심 스텟 레이더
  - 전체 능력치 상세 테이블

### 4. 선수 상세 정보
- 개별 선수 선택 시 상세 능력치 표시
- 포지션별 특화 레이더 차트
- 기술/정신/신체 능력치 세부 정보

## 🚀 실행 방법

### 1. 필요 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 데이터 파일 확인
- `dataset.csv` 파일이 프로젝트 루트에 있는지 확인

### 3. Streamlit 앱 실행

```bash
streamlit run app.py
```

### 4. 웹 브라우저에서 접속
- 자동으로 브라우저가 열립니다 (기본: http://localhost:8501)
- 사이드바에서 필터를 조정하여 원하는 유망주를 찾아보세요!

## 📊 데이터 구조

### 선수 정보
- **기본 정보**: 이름, 나이, 국적, 키, 몸무게
- **기술 능력치** (13개): 드리블, 패스, 슈팅, 태클 등
- **정신 능력치** (14개): 판단력, 예측력, 침착성, 리더십 등
- **신체 능력치** (8개): 속도, 가속력, 체력, 근력 등
- **포지션 숙련도** (15개): 각 포지션별 적합도

## 💡 사용 팁

### 포지션별 유망주 찾기
1. **포지션 선택**: 먼저 원하는 포지션(골키퍼/수비수/미드필더/공격수)을 선택
2. **핵심 스텟 조절**: 해당 포지션에 중요한 능력치 슬라이더가 나타남
3. **기준 설정**: 각 능력치 슬라이더를 조절하여 원하는 최소 기준 설정
   - 예: 공격수 찾기 → 골결정력 15 이상, 스피드 14 이상, 드리블 13 이상
4. **나이 조절**: 유망주는 보통 18-23세가 적합

### 활용 예시
- **빠른 윙어 찾기**: Forward 선택 → 스피드 16+, 드리블 15+, 나이 18-21세
- **강력한 수비수**: Defender 선택 → 태클 15+, 근력 15+, 헤딩 14+
- **패스 마스터 미드필더**: Midfielder 선택 → 패스 16+, 시야 15+, 기술 15+
- **반응 빠른 골키퍼**: Goalkeeper 선택 → 반사신경 15+, 핸들링 14+, 나이 18-24세

### 추가 기능
- **필터 상세보기**: 사이드바에서 현재 적용된 필터 확인
- **능력치 분포**: 선택한 스텟의 분포를 히스토그램으로 확인
- **데이터 다운로드**: 상위 유망주 탭에서 CSV로 다운로드 가능
